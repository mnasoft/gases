* Формулы для расчета влажного воздуа

Теплосодержание влажного воздуха

\( J = 1.005 t + ( 2500 + 1.8 t) \frac{d}{1000} \), кДж/кг

Влагосодержание сжатого воздуха

\( d = 622 \cdot \left( \frac{P_{В.П}}{P_{Б}-P_{В.П}} \right)  \), г/кг

Давление водяных паров

\( {P_{В.П}} = \phi \cdot {P_{В.П.НАС}} \), Па

\phi - относительная влажность;

Давление водяных паров насыщающих воздух

\( {P_{В.П.НАС}} = 133.3 \cdot 10^{\left( \frac {156+8.12t}{236+t} \right)} \), Па
